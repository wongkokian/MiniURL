# MiniURL (https://nodejs-miniurl.herokuapp.com/)

MiniURL is a Node.js web application that generates a 'minimized' URL from a user-provided URL.

# Tech Stack
- Node.js
- Express
- EJS
- MongoDB
- Heroku
